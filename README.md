# 🚀 45 Days #SDESheetChallenge

Hi! 👋

This repository contains my solutions and notes from the **45 Days #SDESheetChallenge**.

I started this challenge to improve my problem-solving skills, strengthen my DSA concepts, and stay consistent with coding practice. My goal is not just to solve problems, but to understand different approaches, optimize solutions, and build a strong foundation for coding interviews.

## Challenge Details

* Start Date: 1 June
* Duration: 45 Days
* Problems Per Day: 3
* Language: Python

## What You'll Find Here

For each day of the challenge, I'll be uploading:

* Problem solutions
* Optimized approaches
* Important observations and learnings
* Code written in Python
